# real-estate-management-system

## Overview

The Real Estate Management System is a Database Management System (DBMS) project designed to manage property listings, clients, agents, sales, and rental transactions efficiently. The system helps streamline real estate operations by maintaining accurate records and improving data management.

## Features

* Property Management
* Client Management
* Agent Management
* Sales Tracking
* Rental Management
* Database Queries and Reports
* Data Integrity and Relationship Management

## Technologies Used

* SQL
* Oracle Database / MySQL (Update according to your project)
* Database Design Concepts
* ERD (Entity Relationship Diagram)
